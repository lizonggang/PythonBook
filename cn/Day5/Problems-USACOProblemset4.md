**问题 1**

```
隐形传输
=============

Farmer John最不喜欢的农业琐事之一就是拖拉
周围有很多牛粪。为了简化这个过程，他
提出了一个辉煌的发明：粪便传送器！代替
他把拖拉机后面的两个点之间的粪便拖到拖拉机后面
可以使用粪便传送器立即从一个运输粪便
位置到另一个。

Farmer John的农场沿着一条长直道建造，所以任何一条
他的农场上的位置可以简单地用它的位置来描述
这条路（实际上是数字线上的一个点）。传送器是
由两个数字x和y描述，其中粪便带到位置x
可以立即运送到位置y，反之亦然。

农民约翰希望将粪便从位置a运输到位置b，
他建造了一个可能在这期间有用的传送器
过程（当然，如果它，他不需要使用传送器
没有帮助）。请帮他确定最低总金额
他需要用拖拉机牵引粪便。

问题名称：传送

输入格式：

第一个也是唯一一个输入行包含四个空格分隔
整数：a和b，描述开始和结束位置，紧随其后
通过x和y，描述传送器。所有职位都是整数
范围0 ... 100，它们不一定是不同的
彼此。

输出格式：

打印一个整数，给出Farmer John所需的最小距离
在拖拉机里拖粪。

样品输入：

3 10 8 2

样本输出：

3

在这个例子中，最好的策略是从中榨取粪便
位置3到位置2，将其传送到位置8，然后将其拖到
因此，需要拖拉机的总距离为1
+ 2 = 3。
```

**问题 2**

```
计数珠子
==============

Bessie已经收集了她的N（1 <= N <= 80）蓝色和橙色的集合
牛头（在人类世界中分别表示为1和0）
地上。她通过将珠子排成长线来清理混乱。
然后，她计算蓝珠与橙色珠子相邻的次数
反之亦然，但不确定她是否正确。写一个程序
验证贝茜的数量。

问题名称：countbead

输入格式：

*第1行：单个整数：N

*第2行：第2行包含N个整数，每个整数为0或1

样品输入：

6
1 0 0 1 1 1

输入细节：

总共有六个珠子：该系列中的第一个珠子是蓝色珠子，
接着是两个橙色珠子，最后是三个蓝色珠子。

输出格式：

*第1行：一个整数，用于描述a的出现次数
        蓝色珠子紧挨着橙色珠子，反之亦然。

样本输出：

2

输出详情：

该系列中有两次接下来是两种不同颜色的珠子
对彼此。
```
