**问题 1**

```
牛计数
============

农夫约翰想要标记他的N（1 <= N <= 1,000,000）奶牛，但是
奶牛不喜欢写上数字L（0 <= L <= 9），
所以他避免这样做。

如果奶牛用最小的N个正整数标记
没有数字L，农民最大的数字是多少
约翰必须写任何一头牛？

问题名称：计数

输入格式：

*第1行：两个以空格分隔的整数：N和L.

样品输入：

10 1

输入细节：

Farmer John有10头奶牛需要在不使用数字1的情况下进行标记。

输出格式：

*第1行：一个整数，是Farmer John的最大数字
         必须写任何一头牛。

样本输出：

22

输出详情：

他可以使用的最小10个数字是：2,3,4,5,6,7,8,9,20和22。
```

**问题 2**

```
开关灯
================

农夫约翰试图通过让他们玩耍来保持奶牛的锋利
智力玩具。其中一个较大的玩具是谷仓里的灯。
N（2 <= N <= 500）奶牛摊位中的每一个都方便地编号
1..N上方有一道彩色灯光。

在晚上的开始，所有的灯都熄灭了。奶牛
使用一组N按钮开关控制灯光
灯光;按开关我改变了关灯的状态
从上到下。

奶牛读取并执行M（1 <= M <= 2,000）次操作的列表
表示为两个整数之一（0 <= operation <= 1）。

第一个操作（由0命令表示）包括两个后续操作
表示起始的整数S_i和E_i（1 <= S_i <= E_i <= N）
开关和结束开关。他们通过推动来执行操作
从S_i到E_i的每个按钮都包含一次。

第二次操作（用1命令表示）要求奶牛计数
在两个整数S_i和S_i给出的范围内有多少个灯亮
E_i（1 <= S_i <= E_i <= N），用于指定包含范围
哪些奶牛应该计算出灯的数量。

帮助FJ确保奶牛通过处理得到正确的答案
列表并产生适当的计数。

问题名称：swtch

输入格式：

*第1行：两个以空格分隔的整数：N和M.

*第2..M + 1行：每行代表一个有三个的操作
        以空格分隔的整数：operation，S_i和E_i

样品输入：
4 5
0 1 2
0 2 4
1 2 3
0 2 4
1 1 4

输入细节：
四盏灯;五个命令。这是应该的顺序
处理：

灯火
            1 2 3 4
  Init：O O O O O = off * = on
  0 1 2  - > * * O O切换指示灯1和2
  0 2 4  - > * O * *
  1 2 3  - > 1计算范围2..3内点亮的灯数
  0 2 4  - > * * O O切换指示灯2,3和4
  1 1 4  - > 2计算1..4范围内点亮的灯数

输出格式：

*第1行......数量的查询：对于每个输出查询，打印计数
        作为一个整数本身在一行上。

样本输出：

1
2
```
