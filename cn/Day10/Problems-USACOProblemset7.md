**问题 1**

```
偶数？ 奇数？
==========

Bessie残忍的二年级老师已经分配了一份N（1 <= N <=
100）Bessie必须确定的正整数I（1 <= I <= 10 ^ 60）
他们的平价（在二年级解释为“偶数......或奇数？”）。 贝茜是
被列表的大小和数字的大小所淹没。 后
所有，她最近才学会数数。

编写程序以读取N个整数并在一行上打印“偶数”
对于偶数而且对于奇数而言同样是“奇数”。

问题名称：偶数

输入格式：

*第1行：单个整数：N

*行2..N + 1：行j + 1包含I_j，第j个整数来确定
        偶/奇

样品输入：

2
1024
5931

输入细节：

两个整数：1024和5931

输出格式：

*行1..N：行j包含单词'even'或'odd'，具体取决于
         I_j的奇偶校验

样本输出：

甚至
奇

输出详情：

1024可以被2整除; 5931不是
```

位）。 因此，她想添加逗号：153,920,529。 请
写一个执行此操作的程序。

输入格式：

样品输入：

153920529

输出格式：

*第1行：在每组三个之前插入逗号的整数N.
         除了第一个数字之外的数字（传统上在许多数字中完成）
        文化）

样本输出：

153920529

**问题 3**

```
字符串函数编码
========================

Bessie发现了一个可以应用于整个牛群的新功能
它的字符串。

给定数字N（1 <= N <= 15）和字符串S，长度
严格大于N，将f（N，S）定义为组成的新字符串
来自字符N的子串的串联（零基于
 - 第一个字符是数字0）到S的结尾和字符串
S本身。

例如，当N = 2，并且S =“COW”时，f（N，S）=“W”+“COW”=
“WCOW”。此外，f（3，“USACO”）=“CO”+“USACO”=“COUSACO”。

Bessie对这个功能很着迷，想要迭代它
几次。例如，如果她为函数迭代一次
“COW”和N = 2，她将获得“WCOW”。如果她应用该功能
当N = 2再次到那个字符串时，她将获得“OWWCOW”，如果她
在N = 2的情况下再次使用它，她将获得“WCOWOWWCOW”。

帮助Bessie编码总共Z（1 <= Z <= 100）个字符串str_1，
str_2，等等。每个str_i的长度在2..100和
仅包含大写字母。每个字符串都带有它
拥有N_i（0 <= N_i <长度（str_i），迭代次数C_i（1 <= C_i）
<= 12）。

问题名称：严厉

输入格式：

*第1行：单个整数：Z

*第2..Z + 1行：第i + 1行包含两个以空格分隔的整数，a
        空格和要编码的字符串：N_i，C_i和str_i

样品输入：

2
2 3 COW
3 2 USACO

输出格式：

*行1..Q：行j包含str_j的迭代编码版本

样本输出：

WCOWOWWCOW
SACOCOUSACO

输出详情：

箭头表示函数的迭代
COW  - > WCOW  - > OWWCOW  - > WCOWOWWCOW
USACO  - > COUSACO  - > SACOCOUSACO
```
