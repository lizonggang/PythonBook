```
机器人犁
==============

农夫约翰购买了一台新的机器人犁以减轻压力
他是从田野后的耕田苦苦挣扎的。它
实现这一目标，但略有不利：机器人犁
只能在具有整数长度边的完美矩形中耕作。

因为FJ的场地有树木和其他障碍，FJ设置了
犁犁许多不同的长方形，这可能最终重叠。
他很好奇他的场地中有多少个正方形
他用各种犁指令对犁进行编程后犁过，
每个都通过给出它的左下角来描述一个矩形
右上角x，y坐标。

像往常一样，场被分割成边的正方形
平行于x和y轴。该场是X平方宽和Y.
正方形高（1 <= X <= 240; 1 <= Y <= 240）。每个I（1 <= I
<= 200）犁指令包括四个整数：Xll，Yll，Xur，
和Yur（1 <= Xll <= Xur; Xll <= Xur <= X; 1 <= Yll <= Yur; Yll <=
Yur <= Y）是左下角和右上角坐标
要耕的长方形。犁将耕种所有的田地
范围内的方块（Xll..Xur，Yll..Yur）可能还有一个
比最初假设的行和列（取决于如何
你当然会考虑你的假设。

考虑一个宽6平方，高4平方的场。作为FJ
发出一对耕作指令（如图所示），该田地被耕种
如'*'和'＃'所示（通常犁过的田地看起来都一样，
但'＃'显示最近被耕种的东西）：

    ......             **....             #####.
    ......  (1,1)(2,4) **....  (1,3)(5,4) #####.
    ......             **....             **....
    ......             **....             **....

共投掷了14个方格。

问题名称：rplow

输入格式：

*第1行：三个以空格分隔的整数：X，Y和I.

*行2..I + 1：行i + 1包含耕作指令i
        由四个整数描述：Xll，Yll，Xur和Yur

样品输入：

6 4 2
1 1 2 4
1 3 5 4

输入细节：

如任务的例子。

输出格式：

*第1行：一个整数，即犁过的总方块数

样本输出：

14
```
